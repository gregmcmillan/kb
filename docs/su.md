# su

Substitute user identity (``su``) is a utility that requests appropriate user credentials and switches to that user ID.

The ``-l`` is a dash followed by a lowercase "L".

```
$ su -l frank 
Password:  
[frank@redd2212 ~]% who 
gmcmillan :0       Dec 14 08:37 
gmcmillan pts/1    Dec 14 09:27 
```
