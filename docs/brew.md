# HomeBrew

It is the missing package manager for macOS.

See http://brew.sh/

## Installation

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Updating

Which version of brew is installed:

```
$ brew --version
Homebrew 5.0.9
```

Update brew to latest version:

```
$ brew update
```
